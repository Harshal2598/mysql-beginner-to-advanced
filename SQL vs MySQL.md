# SQL vs MySQL

## üß© What is SQL?
**SQL (Structured Query Language)** is a **programming language** used to store, manage, and retrieve data from relational databases.

- SQL is a **language**, not software.
- It provides commands like `SELECT`, `INSERT`, `UPDATE`, `DELETE`, etc.
- Used to communicate with relational database systems.

---

## üß© What is MySQL?
**MySQL** is a **Relational Database Management System (RDBMS)** that uses SQL to manage and store data.

- MySQL is **software**, not a language.
- It uses SQL commands to interact with the database.
- Developed by Oracle; open-source and very popular.

---
